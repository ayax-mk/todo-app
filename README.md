Welcome! This is my very first programming project.

 Features
- Add Tasks: Create new todo items.
- View Tasks: See your complete list of tasks.
- Delete Tasks: Remove a specific task by its number.
- Delete All: Clear the entire list with a confirmation safety check ("yes/no").
- Data Persistence:Automatically saves all your tasks into a "tasks.txt" file and reloads them when you restart the app.
- Input Validation: Robust error handling so the app never crashes if you accidentally type letters instead of numbers.

 Technologies Used
- Language: Java 
- Concepts: Arrays (ArrayList), Loops (while), File I/O (FileWriter/Scanner), Exception Handling (try-catch), Conditional Logic (if-else).
